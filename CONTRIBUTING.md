# Contributing to the Washrooms in Vancouver Parks project

This outlines how to propose a change to the Washrooms in Vancouver Parks project.

## Fixing typos

Small fixes to typos or grammar issues in documentation may be edited directly using
the GitHub web interface, so long as the changes are made in the _source_ file.

* YES: you edit a comment in a `.py` or `.md`
* NO: you edit a code portion of a  `.ipynb`

## Prerequisites

Before you make a substantial pull request, you should always file an issue and
make sure someone from the team agrees that it's a problem. If you've found a
bug, create an associated issue and illustrate the bug with a minimal reproducible example.

## Pull request process

* Any larger changes should be made using a new Git Branch for each pull request (PR)
* All new code should follow PEP8 style guide[style guide](https://www.python.org/dev/peps/pep-0008/).

## Code of Conduct

This project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md) and by contributing to or otherwise participating in this project you agree to abide by its terms.
